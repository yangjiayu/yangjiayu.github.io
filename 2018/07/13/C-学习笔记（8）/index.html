C++学习第八章

内联函数

定义： 代码调用是生成相应的计算机语言指令，计算机指令占用不同的内存，不同的函数调用跳到另外内存位置去执行。编译器使用相应的函数代码代替函数调用， 内联函数比一般的函数运行的快，但是占用更多的内存。

使用：

  • 函数声明前加上关键字inline
  • 函数定义前加上关键字inline

引用变量

引用变量主要用途是作为函数的形参

交换两个数的值：

int a = 1;
int b = 2;
swap (&a, &b);

void swap (int a, int b) {
int temp;
temp = a; a = b; b = temp;
}

void print(int a);
void print(int b, int c);

template

void swap(AnyType &a, AnyType &b) {
AnyType temp;
temp = a;
a = b;
b = temp;
}
` 新建立一个模板，类型命名为`AnyType`.关键字`templatetypename是必须的。可以使用简单的类型名T`.

总结

c++通过inline关键字用于函数定义，并在首次调用该函数前提供其函数定义，可以使得C++编译器将该函数视为内联函数。也就是说，编译器不是让程序跳到独立的代码段，以执行函数，而是用相应的代码替换函数调用。只有在函数很短的时候才能采用内联方式。
引用变量是一种伪装指针，允许为变量创建别名。引用变量主要被用作处理结构和类对象的函数的参数。被声明为特定类型引用的标识符只能只想这种类型的数据。然而，如果一个类是从另一个类派生出来的，则基类引用可以指向派生类对象。
C++可以定义函数的默认值，如果函数调用省略了相应的参数，则程序将使用默认值；如果函数调用提供了参数值，程序将使用参数值而不是默认值。只能在参数列表中从右直左提供默认参数，因此，如果为墨某个参数提供了默认值，则必须为该参数右边所有的参数提供默认值。
函数的特征标识其参数列表。可以定义两个同名的函数，只要其特征标不同。这称为函数的重载或者函数多态。
函数模板自动完成重载函数的过程。只需使用泛型和具体算法来定义函数，编译器将为程序中使用的特定参数类型生成正确的函数定义。
